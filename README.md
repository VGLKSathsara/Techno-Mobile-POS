# Techno Mobile POS System

A premium Point of Sale (POS) system for Techno Mobile, designed for retail and repair services..

## Features

- 🔐 Secure Login System
- 💼 Professional Invoice Generation
- 📱 Accessory Management
- 📦 Device & Custom Item Entry
- 💰 Discount Management
- 📜 Invoice History with View/Delete
- 📄 PDF Invoice Export
- 📱 Fully Responsive Design

## How to Use

1. **Login** using the credentials above
2. **Create Invoice:**
   - Enter customer details
   - Select accessories (check boxes)
   - Add devices with storage/IMEI
   - Apply discount if needed
   - Click "Download Invoice"
3. **View History:**
   - Switch to History tab
   - View past invoices
   - Delete individual or all invoices

## Technologies Used

- HTML5
- CSS3 (with CSS Variables)
- JavaScript (ES6+)
- Local Storage for data persistence
- html2pdf.js for PDF generation
- Font Awesome 6 for icons

## File Structure

- `index.html` - Main HTML structure
- `style.css` - All styling
- `script.js` - Core POS functionality
- `history.js` - History management
